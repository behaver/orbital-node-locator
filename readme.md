## 简介

OrbitalNodeLocator 是轨道特殊点位置计算组件库。其中包含有子组件：

* `OrbitalCrossLocator` 轨道交点计算组件

## 安装

通过 npm 安装，在你的 node 项目目录下执行：

`npm install @behaver/orbital-node-locator`

安装完成后，调用即可：

`const { OrbitalNodeLocator } = require('@behaver/orbital-node-locator');`
